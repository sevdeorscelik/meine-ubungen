Notiz

  • Buttons bekommen ein on-click-Event, damit eine Funktion in JavaScript ausgeführt werden kann.
  • Das on-click-Event verlinkte eine Funktion in JavaScript.
  • In der Funktion können HTML-Werte mittels document.getElementById("ID_NAME").innerHTML ausgelesen werden.
  • In der Funktion können Input Values mittels document.getElementById("ID_NAME").value augelesen werden.
